# 概率论的基本概念

## 概率的公理化定义

一个测度，若满足：

1. 非负性：$P(A) \geq 0$；
2. 规范性：$P(S) = 1$；
3. 可列可加性：对 $S$ 中的可列个两两互不相容的事件 $A_1, A_2, \cdots, A_n, \cdots$（即 $A_i A_j = \varnothing, i \not = j, i, j = 1, 2, \cdots$）有
    $$
    P\left(\bigcup_{j=1}^{+\infty} A_{j}\right) = \sum_{j=1}^{+\infty} P\left(A_{j}\right)
    $$

则称 $P(A)$ 为事件 $A$ 发生的概率 (probability)。

## 等可能概型

一个随机试验，若满足：

- 有限性
- 等可能性

则称这个试验为等可能概型，又称古典概型。
