# Neural Networks and Deep Learning

!!! abstract

    Michael Nielsen 的深度学习教程 *Neural Networks and Deep Learning*。

    本书从零开始推导神经网络的核心数学原理，介绍多种改进方式；同时带领读者使用 NumPy 实现 MLP 与 CNN，并在 MNIST 数据集上训练其进行手写数字识别。内容不多，适宜初学。

    - 在线阅读：[Neural networks and deep learning](http://neuralnetworksanddeeplearning.com/index.html)
