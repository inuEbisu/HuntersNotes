# Lec 9. Greedy Algorithms

## Greedy Algorithms

在 optimization problem 中，满足 constraints 的成为 feasible solutions，其中令 optimization function 的值最优的称为 optimal solution。

Greedy methods 一般是 heuristic 的；在满足「局部最优等于全局最优」时正确。

证明范式：

1. 贪心选择性质：可以通过做出局部最优（贪心）选择来构造全局最优解；

2. 最优子结构：一个问题的最优解包含其子问题的最优解。
